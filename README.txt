This program will give you the quarter mile, eighth mile, 60 foot times, as well as the trap speed.

You need to take a datalog of a pull from 0mph, till you've traveled *at least* 1/4 miles


Convert the datalog to a .csv file and upload it through my program.

The following CSV files in this repository are just for testing purposes.
